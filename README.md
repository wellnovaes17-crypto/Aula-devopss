# Projeto Aula DevOps

Este é um projeto simples utilizado para a disciplina de DevOps, com o objetivo de implementar um pipeline completo de **CI/CD** utilizando:

- Git
- GitHub
- GitHub Actions
- GitHub Pages

O pipeline faz:
- ✔️ Integração Contínua (CI)
- ✔️ Deploy Contínuo (CD)
- ✔️ Publicação automática no GitHub Pages
